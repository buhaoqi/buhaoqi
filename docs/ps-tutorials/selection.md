# 选区的创建

## 选区是什么

选区用于分离图像的一个或多个部分。通过选择特定区域，您可以编辑效果和滤镜并将其应用于图像的局部，同时保持未选定 区域不会被改动。

- 使用【修改】命令调整选区

  - [收缩 按特定数量的像素收缩选区

  - [平滑 清除基于颜色的选区中的杂散像素

- 使用【扩展】命令调整扩展选区
  - 【扩大选取】命令：包含所有位于“ 魔棒” 选项中指定的容差范围内的相邻像素。
  - 【选取相似】命令：包含整个图像中位于容差范围内的像素，而不只是相邻的像素。
