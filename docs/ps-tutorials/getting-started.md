# 文档的操作

## 打开图像的方法

- 方法一：**菜单** > **打开**命令
- 方法二：使用快捷键:**Ctrl+O**

## 查看图像的方法

- 使用`抓手`工具查看: 拖动图像以查看图像的其他区域。
  - 双击抓手：使图像适合屏幕大小：
- 使用`缩放`工具查看:
  - 单击放大：选择缩放工具，在图像上单击。
  - 框选放大：选择缩放工具，在图像上框选特定区域，即可放大特定区域。
  - 缩小：选择缩放工具，同时按住Alt键，在图像上单击。
  - 双击缩放工具：按100%比例显示图像
- 使用`导航器`面板查看图像

## 新建图像文件的方法

- 方法一：点击`菜单`,选择`新建`命令
- 方法二：使用快捷键`ctrl+n`

## 关闭图像文件的方法

- 方法一：点图像文件右上角的叉叉
- 方法二：使用快捷键：`ctrl+w`

## 保存图像文件的方法

- 方法一：点击`菜单`,选择`保存`,然后选择你要存储的图像格式，点击`确定`即可
- 方法二：使用快捷键：`ctrl+s`
